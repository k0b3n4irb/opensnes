Detailed Description

Data structure representing one save slot's persistent state.

This struct is serialized directly to/from SRAM as a raw byte block. The SNES has no filesystem – SRAM is a flat byte array, so structs are written at calculated byte offsets. Total size must match SAVE_SIZE.
