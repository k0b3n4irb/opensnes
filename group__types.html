Detailed Description

Integer types with guaranteed sizes.

On the 65816 CPU (cproc compiler):

  • char = 8 bits
  • short = 16 bits
  • int = 32 bits (4 bytes)
  • long = 64 bits (8 bytes — NOT 32-bit!)

Always use these typedefs instead of C primitive types.
