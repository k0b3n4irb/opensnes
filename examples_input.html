Input Examples

The SNES supports standard controllers (12 buttons + D-pad), the SNES Mouse, and the Super Scope light gun. The console reads controller state automatically every frame via auto-joypad read and stores the result in hardware registers.

The key skill is edge detection – distinguishing "button is held" from "button was just pressed this frame." Without it, a single tap registers dozens of times at 60fps.
