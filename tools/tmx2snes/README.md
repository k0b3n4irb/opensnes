# tmx2snes

Converts Tiled map editor JSON exports (.tmj) to SNES binary formats compatible with the OpenSNES map engine.

[Tiled](https://www.mapeditor.org/) is a free, cross-platform level editor. tmx2snes bridges it to the SNES by converting map data, tile properties, and entity positions into binary files that the map engine (`mapLoad`, `mapUpdate`, `mapVblank`) can load directly.

## Usage

```bash
tmx2snes <map.tmj> <tileset.map>
```

- `map.tmj` — Tiled map exported as JSON (.tmj format)
- `tileset.map` — The `.map` file generated by gfx4snes for the tileset image (used for tile index optimization)

## Output Files

| Extension | Content | Map Engine Argument |
|-----------|---------|-------------------|
| `<layer>.m16` | Tilemap data (tile indices + flip bits) | `mapLoad()` arg 1 |
| `<base>.t16` | Tile properties (palette + priority per tile) | `mapLoad()` arg 2 |
| `<base>.b16` | Tile attributes (collision types per tile) | `mapLoad()` arg 3 |
| `<base>.o16` | Entity/object positions and types | `objLoadObjects()` |

The `.m16` filename comes from the Tiled layer name (e.g., layer "BG1" produces `BG1.m16`).

## Example

### 1. Convert tileset with gfx4snes

```bash
gfx4snes -s 8 -o 48 -u 16 -p -m -i tileset.png
# Outputs: tileset.pic, tileset.pal, tileset.map
```

### 2. Convert Tiled map with tmx2snes

```bash
tmx2snes level01.tmj tileset.map
# Outputs: BG1.m16, level01.t16, level01.b16, level01.o16
```

### 3. Use in your game

```c
#include <snes/map.h>

extern u8 mapdata[], tilesetdef[], tilesetatt[];

bgInitTileSet(0, tileset, tilesetpal, 0, size, palsize, BG_16COLORS, 0x2000);
bgSetMapPtr(0, 0x6800, SC_64x32);
mapLoad(mapdata, tilesetdef, tilesetatt);

while (1) {
    mapUpdate();
    mapUpdateCamera(camera_x, camera_y);
    WaitForVBlank();
    mapVblank();
}
```

## Tiled Tile Properties

In the Tiled tileset editor, you can set custom properties on each tile:

| Property | Type | Values | SNES Effect |
|----------|------|--------|-------------|
| `attribute` | string (hex) | `"0000"` = empty, `"FF00"` = solid | Collision type for `mapGetMetaTilesProp()` |
| `palette` | string (hex) | `"0"` to `"7"` | Palette bank (bits 10-12 of tilemap entry) |
| `priority` | string (hex) | `"0"` or `"1"` | BG priority bit (bit 13 of tilemap entry) |

## Tiled Map Requirements

- Tile size: **8x8 pixels** (required by the SNES PPU)
- Max map size: 16384 tiles total (width x height)
- Max height: 256 tiles
- Export format: JSON (.tmj)
- Layer types: "tilelayer" for BG data, "objectgroup" named "Entities" for objects

## Makefile Integration

```makefile
TMX2SNES := $(OPENSNES)/bin/tmx2snes

res/BG1.m16 res/level.t16 res/level.b16: res/level.tmj res/tileset.map
	@cd res && $(TMX2SNES) level.tmj tileset.map
```
