Detailed Description

SuperFX (GSU) coprocessor register definitions and API.

The SuperFX/GSU is a 16-bit RISC coprocessor running at 10.74 or 21.47 MHz. It has its own instruction set (not 65816) and communicates with the main CPU through shared cartridge SRAM and GSU registers at $3000-$303F.

CRITICAL: When the GSU is running, it owns the ROM and RAM buses. The SNES CPU cannot read ROM or SRAM until the GSU stops (STOP instruction).

Function Documentation

◆ gsuInit()

u8 gsuInit ( void  )

Initialize SuperFX — checks if GSU hardware is present.

Returns
1 if GSU detected, 0 if not

The crt0.asm startup code reads VCR and stores the result in superfx_status. This function returns whether the GSU was detected.

Variable Documentation

◆ superfx_status

u8 superfx_status
extern

Detect SuperFX hardware.

Returns
Non-zero if GSU detected (VCR chip version), 0 if not present

Reads VCR register ($303B). Returns the chip version code. On non-SuperFX cartridges, reads open bus (typically $00 or $FF).
